### **Technical Rules & Execution**

* 
**BIM Rules:** The project follows specific standards including **Levels of Development (LOD)**, BIM Dimensions, and **COBie** data exchange criteria.


* 
**Parametric Modeling:** Elements like single beds and railings are created as **Parametric Families**. This allows Revit to store metadata (cost, maintenance) and automatically update all instances across the model when one change is made.


* 
**Conflict Resolution:** A core rule is the use of **Clash Detection** to identify interference—such as a structural column overlapping with furniture—during the design phase to prevent on-site delays.
